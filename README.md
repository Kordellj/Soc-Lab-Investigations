# SOC Lab Investigations Portfolio

This repository documents hands-on SOC investigations performed using:

- ELK Stack (SIEM)
- Splunk
- EDR Concepts
- SOAR Workflows

Each investigation includes:
- Scenario description
- Log analysis methodology
- Detection logic
- MITRE ATT&CK mapping
- Analyst response actions

Endpoint detection and response analysis workflows.

### SOAR Investigations
Automation and incident response playbook documentation.
